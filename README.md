Installation
================

1: Upload you files to server root.
<br />
2: Point your browse to http://www.yourdomain.ro and follow the installation steps (yourdomain.ro change it to your real domain). 

================

Requirements
================

1: PHP 5.3 > Above
<br />
2: Apache Web Server 2 > Above or NGINX
<br />
3: MySql Server
<br />
4: MySqli Extension
<br />
5: zlib
<br />
6: mbstring
<br />
7: gd2
